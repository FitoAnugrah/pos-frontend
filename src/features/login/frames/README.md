# Login Session Frames

Struktur ini dipakai untuk sesi login berdasarkan frame Figma. Untuk saat ini, implementasi UI berhenti di frame pertama.

1. `frame-01-lupa-kata-sandi-pos-ai`
   Status: UI + text selesai.
2. `frame-02-buat-akun-pos-ai`
   Status: text siap, UI belum dibuat.
3. `frame-03-verifikasi-otp-pos-ai`
   Status: text siap, UI belum dibuat.
4. `frame-04-masuk-pos-ai`
   Status: text siap, UI belum dibuat.
5. `frame-05-verifikasi-otp-alternatif-pos-ai`
   Status: text siap, UI belum dibuat.
6. `frame-06-kata-sandi-baru-pos-ai`
   Status: text siap, UI belum dibuat.

Catatan:
- Nama folder mengikuti nama frame pada Figma.
- Setiap frame menyimpan string konten di `text.js`.
- `src/App.jsx` saat ini hanya merender frame 1 (`Lupa Kata Sandi`).
